# hyrox备赛指南 全流程陪伴你hyrox备赛-从萌新到站台 🏋️

> HYROX健身赛事全流程备赛助手，从萌新门到站台一站式指导

## 📖 简介

HYROX备赛指南是一个专业的技能(Skill)，帮助HYROX赛事参与者进行科学备赛。无论你是新手还是老手，都能获得个性化的备赛支持。

我是一名亲身参与过三届HYROX的普通爱好者。从最初在场边为队友加油，到后来站上接力赛道完赛，这个比赛让我真正体验到运动的快乐，也真切体会过普通人备赛时的迷茫与无助。
虽然我的运动基础普通，跑两步心率便容易飙升，赛场表现也算不上亮眼，但我始终相信：**8公里8个站是赛场，赛场之外，依然有可以发光的地方**。
于是我把自己的参赛经验、训练思考和信息整理能力，做成了这个开源备赛助手，希望用热爱帮助更多人少走弯路，顺利站上HYROX赛场。

## ✨ 核心功能

### 🚀 新手友好
- 7天快速体验计划，无需填信息直接开始
- 5分钟了解HYROX是什么
- 3个问题生成个性化备赛计划

### 💪 科学训练
- 入门/进阶/精英分级方案
- 8个功能站极简动作要点
- 疲劳链训练，模拟真实比赛场景
- 动态调整规则，根据状态自动优化

### 👥 团队赛支持
- 四人接力自动分工匹配
- 双人组配合策略
- 陌生人组队识别技巧

### 🏥 全场景覆盖
- 伤病替代动作库
- 女性/40+年龄组适配
- 国内替代训练方案（无器械也能练）
- 比赛日紧急处理指南

### 🏆 赛后闭环
- 成绩分析报告
- 成绩预测模型
- 下一届备赛规划
