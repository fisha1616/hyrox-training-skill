# HYROX官方规则详解

> 基于HYROX官方FAQ和2025/26赛季规则整理

---

## 1. 参赛资格

### 年龄要求
- **最低年龄**：16岁
- **最高年龄**：无上限（已知最年长参赛者80岁）
- **年龄计算**：按比赛当天（day of the race）的实际年龄计算

### 资格要求
- **无需任何资格认证**
- **无完赛时间限制**（会等到最后一位选手安全完赛）
- **完赛率**：98%的参赛者都能完赛

## 2. 分组详解

### 四大组别

| 组别 | 特点 | 适合人群 |
|------|------|----------|
| 个人组 Individual | 男子Open / 男子Pro / 女子Open / 女子Pro | 无混合组 |
| 双人组 Doubles | 男子 / 女子 / 混合 | 混合是一男一女 |
| 接力组 Relay | 男子 / 女子 / 混合 | 4人接力 |

### 年龄组划分

**单人组/双人组（14个年龄组）**：
- (a) 24岁及以下（16-24岁）
- (b) 25-29岁 / (c) 30-34岁 / (d) 35-39岁
- (e) 40-44岁 / (f) 45-49岁 / (g) 50-54岁 / (h) 55-59岁
- (i) 60-64岁* / (j) 65-69岁* / (k) 70-74岁* / (l) 75-79岁*
- (m) 80-84岁* / (n) 85-89岁*

> ⚠️ *精英组年龄上限为(h)55-59岁组，60+选手不具备精英组参赛资格

**接力组（2个年龄组）**：
- U40（16-39岁）：按四名队员平均年龄划分
- 40+

## 4. 罚时规则

### 时间罚时

| 违规类型 | 罚时 |
|----------|------|
| 漏跑圈数 | 3-7分钟/圈（具体由场地决定） |
| 站点顺序错误 | 3分钟 |
| 多个站点顺序错误 | 取消资格 |
| 漏掉整个站点/跑步 | 取消资格 |
| Roxzone进出口错误 | 2分钟/次 |
| 吐痰/擤鼻涕到地面 | 2分钟/次 |

### 距离/时间罚时（站点1-7）
- 每个站点**先给一次警告**
- 第二次警告：动作无效 + 罚时
- 之后每次违规继续罚时

### 未完赛/取消资格
- **DNF（Did Not Finish）**：无法获得成绩和排名，但可继续比赛
- **DQ（Disqualified）**：被取消资格，必须停止比赛

## 5. 比赛流程

### 赛前准备

| 时间 | 事项 |
|------|------|
| 赛前1周 | 公布出发时间表 |
| 赛前1.5-1小时 | 到达场馆、签到、寄存物品 |
| 出发前10分钟 | 到达起跑区 |

### 签到流程
1. 出示**身份证件** + **报名二维码**
2. 获得选手手环
3. 佩戴计时芯片（右脚踝）
4. 号码写在前臂

### 存包须知
- 允许寄存小型运动包
- 完赛后**1小时内**取回
- 不负责存包区外物品

---

## 6. 补给规定

### 规则
- 所有补给必须**从起点携带**
- Roxzone每次经过都有**水**
- 运动饮料详情见《参赛手册》
- **禁止接受场外补给**（视为外部援助，可能罚时或取消资格）

### 建议
- 完赛<90分钟：可能不需要中途补给
- 完赛90-120分钟：建议在Roxzone放水
- 完赛>120分钟：需要积极补给策略

## 10. 世界锦标赛与精英15强

### 世锦赛资格获取

| 途径 | 说明 |
|------|------|
| **大满贯赛事** | 汉堡、墨尔本、菲尼克斯、华沙（共4场），前3名直接获得世锦赛资格 |
| **区域锦标赛** | 华盛顿（美洲）、哥本哈根（EMEA）、布里斯班（亚太），冠军直接获得资格 |
| **常规赛** | ❌ 不能直接获得世锦赛资格 |

> ⚠️ 接力组无世锦赛名额

### 精英15强（Pro 15）晋级体系

**什么是精英15强？**
- 专为男女精英组全球最优秀15名选手设置的独立出发组别
- 只有精英15强选手有资格角逐"HYROX健身跑世界冠军"并赢取奖金

**晋级计算方式：**
- 过去365天内**2次最快成绩的平均值**（动态排名，每天更新）
- 大满贯每场确保3男3女获得资格（名额顺延）

**关键限制：**
- 精英15强选手**不参与世锦赛个人年龄组比赛**
- 晋级通知通过邮件发送，逾期未注册名额顺延

### 60+年龄组特殊规定

| 规定 | 说明 |
|------|------|
| 精英组资格 | ❌ 60岁及以上不符合精英组参赛资格 |
| 世锦赛负重 | 保持原组别标准不变（不升重） |
| 精英组上限 | 55-59岁组 |

## 11. 区域锦标赛

### 三大区域划分

| 区域 | 覆盖范围 |
|------|----------|
| **美洲区** | 北美、南美国家 |
| **亚太区（APAC）** | 中国、日本、韩国、澳大利亚等 |
| **欧洲、中东和非洲区（EMEA）** | 欧洲、中东、非洲国家 |

### 参赛规则
- 所有组别采用**大众组负重标准**
- 参赛资格：必须为该区域国家的**公民或合法永久居民**
- 身份确认：需政府签发的有效身份证件（护照或永久居留卡）
- **不接受**驾驶执照或复印件
