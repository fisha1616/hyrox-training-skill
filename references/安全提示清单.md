# 安全提示清单

> HYROX备赛和比赛涉及高强度训练，安全永远是第一优先级。本清单帮助您**识别风险、预防伤害、科学恢复**。

---

## 一、疲劳状态下动作风险提示

### ⚠️ 高危信号（必须停止）

| 信号 | 风险 | 立即行动 |
|------|------|----------|
| 动作变形明显 | 受伤风险↑300% | 停止该动作 |
| 关节周围刺痛 | 软组织损伤 | 停止训练 |
| 头晕/恶心 | 过度疲劳/脱水 | 坐下休息+补水 |
| 呼吸急促无法说话 | 心肺过载 | 降速/停止 |

### 疲劳状态下常见危险动作

| 动作类型 | 危险原因 | 疲劳时风险 |
|----------|----------|------------|
| **深蹲跳** | 膝关节冲击 | 落地不稳→踝/膝扭伤 |
| **农夫行走** | 脊柱压力 | 核心失稳→腰损 |
| **划船** | 下背弓背 | 腰椎间盘压力↑ |
| **墙球** | 脊柱旋转 | 椎间盘损伤 |

### 安全原则
```
疲劳时三不原则：
❌ 不做跳跃类高冲击动作
❌ 不做大重量训练
❌ 不做憋气用力动作
```

## 二、伤病停止训练红线

### 🚫 必须停止训练的信号

| 症状 | 可能问题 | 建议 |
|------|----------|------|
| **关节锐痛** | 韧带/软骨损伤 | 立即停止，48小时内就医 |
| **胸痛/胸闷** | 心脏问题 | 立即停止，呼叫急救 |
| **呼吸困难** | 心肺问题 | 立即停止 |
| **肌肉剧痛** | 肌肉撕裂 | 停止，RICE处理 |
| **头晕/视物模糊** | 神经系统/脱水 | 停止，坐下补水 |
| **关节肿胀** | 炎症/损伤 | 停止，冰敷就医 |
| **持续疼痛>3天** | 过度使用损伤 | 停止训练，咨询医生 |

### 可以继续训练的情况

| 情况 | 说明 | 调整方法 |
|------|------|----------|
| 肌肉酸痛（DOMS） | 训练后24-72小时正常 | 降低强度继续 |
| 轻微疲劳感 | 正常适应反应 | 减量训练 |
| 关节不适（非锐痛） | 轻度劳损 | 换动作/减重量 |

### 自我评估方法

```
疼痛等级（0-10）：

0-2分：✅ 可以训练（轻微不适）
3-4分：⚠️ 降低强度，调整动作
5-6分：🔴 停止该项目，休息
7分以上：🚫 立即停止，48小时就医
```

## 三、训练前热身检查清单

### 🔥 热身流程（10-15分钟）

| 顺序 | 内容 | 时间 | 要点 |
|------|------|------|------|
| 1 | **慢速有氧** | 3-5分钟 | 轻松慢跑/划船机 |
| 2 | **动态拉伸** | 5分钟 | 腿部+髋关节+肩部 |
| 3 | **动作激活** | 3-5分钟 | 徒手深蹲、踢臀等 |
| 4 | **强度递增** | 2分钟 | 逐步提高心率 |

### 动态拉伸清单

```
✅ 下肢激活（每个动作10-15次）：
├─ 高抬腿走
├─ 踢臀跑
├─ 侧向弓步
├─ 臀推行走
└─ 腿摆动（前后+左右）

✅ 上肢激活（每个动作10次）：
├─ 手臂画圈
├─ 肩部环绕
├─ 胸椎旋转
└─ 猫牛式伸展

✅ 核心激活（每个动作30秒）：
├─ 平板支撑
├─ 四点支撑交替抬手
└─ 死虫动作
```

### ⚠️ 热身不充分的后果
- 肌肉拉伤风险↑ 60%
- 关节损伤风险↑ 40%
- 运动表现下降 15-20%

## 四、训练后恢复建议

### 恢复金字塔

```
        ┌─────────┐
        │  睡眠   │  ← 最关键（7-9小时）
        ├─────────┤
        │  营养   │  ← 蛋白质+碳水
        ├─────────┤
        │  主动   │  ← 轻松活动/拉伸
        │  恢复   │
        ├─────────┤
        │  被动   │  ← 泡沫轴/按摩/冰浴
        │  恢复   │
        └─────────┘
```

### 训练后恢复清单

| 项目 | 内容 | 时间 |
|------|------|------|
| **冷身** | 5分钟轻松有氧 + 拉伸 | 训练后立即 |
| **补水** | 体重丢失量的150% | 训练后2小时内 |
| **营养** | 蛋白质20-30g + 碳水 | 训练后30分钟内 |
| **泡沫轴** | 重点肌群滚动 | 训练后30分钟 |
| **睡眠** | 7-9小时高质量睡眠 | 当晚 |

### 肌肉酸痛处理

| 方法 | 适用情况 | 效果 |
|------|----------|------|
| **主动恢复** | 轻微酸痛 | 促进血液循环 |
| **泡沫轴** | 筋膜紧张 | 放松扳机点 |
| **冰敷** | 急性炎症/肿胀 | 减轻疼痛 |
| **热敷** | 慢性酸痛 | 促进恢复 |
| **按摩** | 深度放松 | 筋膜松解 |

### ⚠️ 恢复不足的信号
- 持续疲劳超过3天
- 静息心率持续偏高（+10bpm）
- 运动表现持续下降
- 情绪低落/易怒

## 六、明确不提供的内容

> 以下内容**超出本Skill的能力范围**，我们不会提供，请理解：

### 🚫 我们不提供

| 类型 | 原因 | 建议 |
|------|------|------|
| **医疗诊断** | 非医疗专业人士 | 请咨询医生 |
| **线下教练替代** | 无法提供实时指导 | 建议聘请专业教练 |

| **心脏疾病评估** | 需要专业检查 | 请咨询心脏科医生 |

### 为什么不能提供？

```
1. 每个人身体情况不同
2. 需要专业评估才能给出建议
3. 错误建议可能造成严重后果
4. 医疗行为需要资质
```

### 遇到这些情况怎么办？

| 情况 | 正确做法 |
|------|----------|
| 伤痛持续 | 预约运动医学科/骨科 |

| 营养问题 | 咨询注册营养师 |
