## 📋 7项自测清单

> ⚠️ **测试前准备**
> - 穿着运动服装和跑鞋
> - 空腹或餐后2小时
> - 准备：秒表、跑步场地、哑铃/重物

## 1️⃣ 心肺耐力测试

### 选项A：3分钟台阶测试（家庭版）
```
📋 器材：稳固台阶或矮凳，高度30-40cm
📋 音乐：节拍器120bpm（或自己喊节奏）

动作：
1. 单脚上台阶，另一脚跟上
2. 双脚站稳
3. 换脚，循环3分钟

记录：
- 完成后心率：___ bpm
- 完成后呼吸恢复时间：___秒
```

| 评分 | 心率恢复时间 | 解读 |
|:----:|--------------|------|
| 1分 | >2分钟 | 心肺急需加强 |
| 2分 | 90-120秒 | 有氧基础薄弱 |
| 3分 | 60-90秒 | 基础尚可 |
| 4分 | 30-60秒 | 有氧能力良好 |
| 5分 | <30秒 | 心肺优秀 |

### 选项B：1.6km跑（操场版）
```
📋 场地：400m跑道4圈
📋 计时：完成时间

记录：___分___秒

评分标准（男性/女性）：
| 评分 | 男性 | 女性 |
|:----:|------|------|
| 1分 | >9:00 | >11:00 |
| 2分 | 8:00-9:00 | 9:30-11:00 |
| 3分 | 7:00-8:00 | 8:00-9:30 |
| 4分 | 6:00-7:00 | 7:00-8:00 |
| 5分 | <6:00 | <7:00 |
```

## 2️⃣ 下肢力量测试

```
📋 动作：双脚与肩同宽，下蹲至大腿与地面平行
📋 要求：膝盖不内扣，腰背挺直
📋 记录：连续完成次数

评分标准：
| 评分 | 次数（男/女） |

| 2分 | 10-20 / 5-10 |
| 3分 | 21-30 / 11-20 |
| 4分 | 31-40 / 21-30 |
| 5分 | >40 / >30 |
```

## 📊 雷达图生成指南

### 5维度分数汇总

| 维度 | 测试项目 | 得分(1-5) |
|------|----------|:---------:|
| **心肺耐力** | 1.6km跑 / 台阶测试 | __ |
| **下肢力量** | 深蹲 + 保加利亚蹲 | __ |
| **握力** | 悬垂 / 引体向上 | __ |
| **核心稳定** | 平板支撑 + 单腿站 | __ |
| **技术水平** | SkiErg / 壶铃摇摆 | __ |
| **抗疲劳** | 连续循环测试 | __ |

> **注：抗疲劳能力不单独出现在雷达图，但会影响你的训练优先级判断**

### 雷达图模板

```
                    5分
                     │
                     │          4分
            ┌────────┼────────┐
            │   心肺  │        │
            │  耐力   │        │
     4分────┤  (__)  │        ├────4分
            │        │        │
            │ 握力   │  下肢  │   水平
     3分────┤ (__)   │  力量  ├────3分
            │        │ (__)   │
            │        │        │
            └────────┼────────┘
                     │
                     │          2分
                    1分
                    │
              核心稳定
               (__)

实际得分标注：● 心肺耐力  ● 下肢力量  ● 握力
              ● 核心稳定  ● 技术水平
```

### 如何绘制（手绘或电子版）

1. **画五边形网格**：5层同心五边形
2. **标注五个顶点**：心肺耐力、下肢力量、握力、核心稳定、技术水平
3. **标注分数**：在每个轴上从中心点向外的第1-5层分别代表1-5分
4. **描点连线**：将你的5项得分用线连接
5. **填充颜色**：填充区域代表你的能力轮廓

## 🎯 根据雷达图的站点选择建议

### 典型雷达图类型

#### 类型A：跑步型 ⛹️
```
特征：心肺耐力4-5，其他2-3
表现：跑步轻松，站点容易崩

→ 训练重点：站点专项 + 抗疲劳
→ 推荐优先练：墙球、滑雪机、农夫行走
```

#### 类型B：力量型 🏋️
```
特征：下肢/握力4-5，心肺2-3
表现：站姿没问题，跑步先喘

→ 训练重点：跑步间歇 + 配速控制
→ 推荐优先练：Zone 2慢跑、400m间歇
```

#### 类型C：均衡型 ⚖️
```
特征：全部3-4分
表现：都能应付，但没有亮点

→ 训练重点：查漏补缺 + 抗疲劳
→ 找出最低分项重点突破
```

#### 类型D：新手型 🌱
```
特征：全部1-2分
表现：需要全面提升

→ 训练重点：基础体能 + 习惯养成
→ 从12周计划第1周开始
```

#### 类型E：某一项极低型 🚨
```
特征：某项1分，其他3+
表现：那一项会拖垮整体节奏

→ 训练重点：**必须专项突破最低项**
→ 例如：握力1分 → 每天加入农夫行走和悬垂
```
