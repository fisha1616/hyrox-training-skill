# HYROX新手入门指南

> 如果你是第一次听说HYROX，这篇文章将用**5分钟**让你了解这项赛事，并做出是否参赛的决定。

## 🚀 5分钟快速了解HYROX

### HYROX是什么？

**HYROX = 跑步 + 功能性健身的标准化竞速赛事**

想象一下：你要完成**8公里跑**，但每跑完1公里，就要停下来完成一个健身动作站。总共8个站点，全程约60-120分钟。

```
🏃 1km跑 → 🏋️ 站点1 → 🏃 1km跑 → 🏋️ 站点2 → ... → 🏃 1km跑 → 🏋️ 站点8 → 🎉 终点
```

### 为什么HYROX这么火？

| 特点 | 说明 |
|------|------|
| **全球统一** | 无论在汉堡、迈阿密还是上海，赛道完全相同 |
| **公平竞争** | 不分职业/业余，统一分组按年龄排名 |
| **门槛适中** | 16岁以上、有运动基础即可参加 |
| **成就感强** | 完赛奖牌、详细成绩单（各站分段时间） |
| **社群氛围** | 全球超过10万运动员的社区 |

### HYROX适合谁？

- ✅ **有跑步习惯**的人（能跑5-10公里）
- ✅ **CrossFit/健身爱好者**想测试综合体能
- ✅ **追求新挑战**的运动玩家
- ✅ **想设立目标**并为之训练的人

## 📊 组别选择自测表

### Open vs Pro 怎么选？

| 问题 | 选 Open ✅ | 选 Pro ✅ |
|------|-----------|-----------|
| **你有规律训练习惯吗？** | 少于2年 | 2年以上 |
| **你经常做功能性训练吗？** | 偶尔/不 | 每周2次以上 |
| **你参加过一次HYROX吗？** | 否 | 是 |
| **你的目标是什么？** | 完赛/挑战自我 | 争取名次 |
| **你能推动多重雪橇？** | 102kg有点吃力 | 202kg可以尝试 |

### 快速决策

> **一句话判断**：如果这是你第一次参加HYROX，**无脑选Open**。
> 
> Pro的雪橇重量是Open的**2倍**，新手很可能完不成。

### 年龄组说明

- **U24**：16-23岁（最年轻组）
- **25-29、30-34...**：每5岁一组
- **70+**：最年长组
- 选择依据：完赛年份12月31日前的年龄

## ⏱️ 时间投入预期

### 12周备赛需要多少时间？

| 阶段 | 周数 | 每周训练天数 | 单次时长 | 总计 |
|------|------|--------------|----------|------|
| **基础期** | 1-4周 | 4天 | 40-60分钟 | ~16小时 |
| **专项期** | 5-8周 | 5天 | 45-75分钟 | ~24小时 |
| **竞赛期** | 9-11周 | 5天 | 50-80分钟 | ~22小时 |
| **减量期** | 第12周 | 3天 | 20-40分钟 | ~3小时 |
| **合计** | 12周 | - | - | **~65小时** |

### 每周典型训练安排

```
周一：轻松跑 30-40分钟 🏃
周二：站点训练 45-60分钟 🏋️
周三：休息或轻度拉伸 💤
周四：间歇跑 + 站点练习 40分钟 🏃🏋️
周五：休息 💤
周六：长跑或模拟训练 60-75分钟 🏃🏋️🏃
周日：休息 💤
```

### 你需要的基础

| 能力 | 最低要求 | 推荐水平 |
|------|----------|----------|
| **跑步** | 能连续跑5公里 | 能跑10公里 |
| **力量** | 推动自身重量 | 推动1倍体重 |
| **深蹲** | 能做20个标准深蹲 | 能做50个 |
| **波比跳** | 能做5个连续标准动作 | 能做15个连续 |

> 💡 **自测**：去健身房试一次完整流程（1km跑+8站点），如果能坚持下来，你就可以开始备赛。

## 📋 第一次参赛须知

### 赛前1周

- [ ] 确认比赛时间和地点
- [ ] 规划出行路线（提前1.5小时到达）
- [ ] 准备好比赛服装（提前试穿）
- [ ] 准备早餐食物（熟悉、不会肠胃不适）
- [ ] **减量训练**：减少训练量，保持活跃
- [ ] 早睡早起，调整作息

### 比赛日流程

| 时间 | 行动 | 备注 |
|------|------|------|
| **起床** | 赛前2.5-3小时 | 喝500ml水 |
| **早餐** | 赛前2小时 | 碳水为主，低脂低纤维 |
| **到达** | 赛前90分钟 | 领取物资、熟悉场地 |
| **检录** | 赛前60分钟 | 进入场地 |
| **热身** | 赛前30分钟 | 轻度跑步+动态拉伸 |
| **出发** | 按你的批次 | 全力以赴！ |
| **完赛** | 领取奖牌+成绩单 | 享受荣耀时刻 |

### 比赛当天带什么？

| 物品 | 备注 |
|------|------|
| **号码布** | 提前别在比赛服上 |
| **跑鞋** | 熟悉的、已穿开的 |
| **运动服** | 速干、不限制活动 |
| **水壶/运动饮料** | 根据预计完赛时间 |
| **能量胶** | 90分钟以上完赛建议带 |
| **毛巾** | 擦汗、垫坐 |
| **凡士林** | 涂抹易摩擦部位 |
| **替换衣服** | 赛后换穿 |

### 赛中策略（新手版）

| 原则 | 说明 |
|------|------|
| **前慢后稳** | 前1km刻意保守，不要被带节奏 |
| **保持呼吸** | 任何时候都不要憋气 |
| **允许休息** | 波比跳/墙球可以分组完成，不丢脸 |
| **享受过程** | 第一次参赛，完赛就是胜利！ |

### 赛后恢复

| 时间 | 行动 |
|------|------|
| **0-30分钟** | 领取奖牌、慢走放松、补充蛋白质+碳水 |
| **2-3小时** | 正式餐食、充足蛋白质 |
| **当天晚上** | 拉伸、轻度泡沫轴、充足睡眠 |
| **接下来3天** | 主动恢复（散步、拉伸）、高蛋白饮食 |
| **接下来2周** | 逐渐恢复正常训练强度 |

## 🎯 新手常见问题速答

**Q：需要提前多久开始备赛？**
> 推荐**12周**，但有运动基础的人6-8周也可以。

**Q：Open组完赛时间大概多长？**
> 男子平均90-110分钟，女子平均100-120分钟。

**Q：可以现场报名吗？**
> 部分赛事有现场报名，但**强烈建议提前报名**，名额有限。

**Q：比赛中受伤怎么办？**
> 立即停止，向裁判报告。完赛不是最重要的，身体健康才是。
