# HYROX 核心规则速查

> 最后更新：2026-04-13

## 一、组别分工规则（最易混淆）

### 双人组（Doubles）

| 项目 | 规则 | 常见误区 |
|------|------|----------|
| **跑步** | 两人都跑完全程**8km**，必须保持2米内 | ❌ 每人跑4km |
| **力量站** | **非常灵活**，每个站点可随意按组数/次数/轮换分工 | ❌ 每人固定做4个站 |
| **同时工作** | ❌ 禁止，一次只能一人工作，另一人休息 | - |
| **总工作量** | 与单人相同（如划船机总共1000m） | - |
| **罚时** | 跑步差距>15秒罚**1分钟** | 累计>3次取消排名 |

### 双人组器材传递规则（易犯规）

| 器材 | 传递规则 | 违规后果 |
|------|----------|----------|
| 滑雪机手柄 | ❌ 严禁直接传递 | 罚时 |
| 划船机手柄 | ❌ 严禁互相传递或调节阻力阀 | 罚时 |
| 壶铃 | ✅ 仅允许横向或向后传递，❌ 禁止向前传递 | 罚时 |
| 沙袋 | ✅ 仅允许横向或向后传递，❌ 禁止向前传递 | 罚时 |
| 墙球 | ✅ 仅允许地面交接或手对手传递，❌ 禁止空中传递 | 罚时 |

> ⚠️ 同一时间仅允许一人工作，另一人必须在指定休息区等待

### 接力组（Relay）

| 项目 | 规则 |
|------|------|
| **分工** | 4人团队，每人完成**2km跑 + 2个功能站** |
| **分组** | 按平均年龄：U40（16-39岁）/ 40+（40岁以上） |
| **世锦赛** | ❌ 接力组无世锦赛名额 |

## 二、雪橇距离（最易标错）

| 站点 | 单程距离 | 实际执行 | 说明 |
|------|----------|----------|------|
| **推雪橇** | 50m | 往返4趟 = 200m | 每趟推过12.5m线后转身 |
| **拉雪橇** | 50m | 往返4趟 = 200m | 同上 |

> ⚠️ 注意：标题写作"50m"，但实际是往返4趟，不是"50m×2"或"25m×4"

## 三、罚时规则

| 违规类型 | 罚时 | 备注 |
|----------|------|------|
| 漏跑圈数 | **3-7分钟/圈** | 具体由场地决定 |
| 站点顺序错误 | **3分钟** | - |
| 多个站点顺序错误 | **取消资格** | - |
| 漏掉整个站点/跑步 | **取消资格** | - |
| Roxzone进出口错误 | **2分钟/次** | - |
| 吐痰/擤鼻涕到地面 | **2分钟/次** | - |
| 防滑粉违规使用 | **2分钟/次** | 单人：仅指定站点；双人：仅推雪橇和农夫行走 |
| 双人跑步分离>15秒 | **1分钟** | 累计>3次取消排名 |

## 五、手机与拍摄规定

| 规定 | 说明 |
|------|------|
| **手机** | ❌ 严禁携带，违规取消资格 |
| **耳机** | ❌ 严禁携带 |
| **GoPro/运动相机** | ❌ 严禁携带 |
| **允许携带** | 护膝、手套、举重腰带、腕带、水袋包、小型医疗用品 |

### 拍照解决方案
1. 官方摄影师Sportograf：赛后约48小时可查看下载
2. 观众席拍摄：朋友在观众区帮忙
3. 预约跟拍服务：现场购买
