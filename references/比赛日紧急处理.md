# 比赛日紧急处理指南

> HYROX比赛强度大、持续时间长（60-90分钟），比赛中可能遇到各种突发状况。本指南帮助你在关键时刻**正确应对**，安全完赛。

---

## 一、抽筋处理

### 识别症状
- 肌肉突然剧烈疼痛、僵硬
- 局部肌肉鼓起、变硬
- 无法继续当前动作

### 现场处理步骤

| 步骤 | 操作 | 时间 |
|------|------|------|
| 1️⃣ | **停止当前动作**，缓慢移至安全区域 | 立即 |
| 2️⃣ | **轻柔拉伸**抽筋部位（不要猛然拉伸） | 30-60秒 |
| 3️⃣ | **反向按压**肌肉痛点，持续施压 | 15-30秒 |
| 4️⃣ | **补充电解质**（盐丸/运动饮料） | 吞服 |
| 5️⃣ | **轻揉放松**后，小幅度活动 | 1-2分钟 |

### 预防措施
- 赛前30分钟补充**盐丸1-2颗**
- 每15-20分钟小口喝**电解质饮料**
- 大量出汗后及时补充**钠、钾、镁**

### 可用补给推荐
- 盐丸（康比特、GU、SIS）
- 电解质泡腾片
- 香蕉（提前吃）

## 二、低血糖处理

### 识别症状
- 头晕、眼花、视线模糊
- 手抖、出冷汗
- 饥饿感强烈
- 思维迟钝、注意力下降
- 面色苍白

### 快速补充方案

| 紧急程度 | 推荐食物 | 吸收时间 | 适用场景 |
|----------|----------|----------|----------|
| 🔴 紧急 | **葡萄糖凝胶/胶** | 5-10分钟 | 正在比赛中 |
| 🟡 较急 | **能量胶 + 水** | 10-15分钟 | 功能站间隙 |
| 🟢 预防 | **香蕉/能量棒** | 15-20分钟 | 赛前1小时 |

### 处理流程
```
1. 感知不适 → 立即告知身边选手/志愿者
2. 移至路边 → 避免阻挡赛道
3. 补充糖分 → 优先凝胶，后固体食物
4. 等待恢复 → 血糖回升后再出发
5. 降低配速 → 完赛后补充正餐
```

### ⚠️ 警告
- **不要空腹参赛**
- 比赛前2小时吃易消化碳水（面包、香蕉）
- 高血糖≠好事，不要过量补糖

## 三、装备故障

### 常见故障与应对

| 故障类型 | 紧急程度 | 现场处理 | 后续建议 |
|----------|----------|----------|----------|
| **鞋带断裂** | 🔴高 | 换备用鞋/系死结/用胶带绑 | 赛前检查，备双鞋 |
| **鞋垫跑偏** | 🟡中 | 暂停重新塞好 | 选择紧致跑鞋 |
| **护具脱落** | 🔴高 | 暂停固定后再出发 | 检查松紧后再参赛 |
| **手套破损** | 🟢低 | 继续比赛/换备用 | 多带一副 |
| **手表脱落** | 🟢低 | 暂停戴好再继续 | 检查表带 |

### 鞋带断裂紧急处理
```
方案1: 备用鞋（推荐放换项区）
方案2: 鞋带打死结继续跑
方案3: 用创可贴/胶带缠绕固定
方案4: 寻求志愿者帮助
```

### 护具（护膝/护踝）脱落
```
处理原则：必须固定好再继续
1. 暂停于安全区域
2. 重新穿戴/固定
3. 检查松紧度
4. 小跑测试稳定性
5. 确认无误后继续
```

## 四、体力透支处理

### 识别体力透支信号
- 心率持续过高（>180bpm）
- 配速明显下降（>30秒/公里）
- 呼吸急促、喘不过气
- 腿发软、脚步变拖
- 恶心想吐

### 降速策略

| 阶段 | 策略 | 目标 |
|------|------|------|
| 初期透支 | 走+小跑交替 | 保持运动状态 |
| 中度透支 | 降速20-30% | 保存体力完赛 |
| 严重透支 | 走完本功能站 | 到补给站再决定 |

### 补给调整
- **增加频率**：每站后小口补水
- **补充碳水**：能量胶/运动饮料
- **电解质加强**：盐丸加倍
- **暂停休息**：情况严重就停下来

### 分段完赛策略
```
如果真的跑不动了：
├─ 允许自己走路通过功能站
├─ 每站后喝一口水
├─ 设定"走到下一个补给点"的小目标
└─ 完赛就是胜利！
```

## 五、心理崩溃应对

### 常见心理危机信号
- "我不行了" 的念头反复出现
- 对剩余距离感到绝望
- 身体疼痛被放大
- 极度想放弃

### 鼓励话术（自我对话）

| 场景 | 自助话术 |
|------|----------|
| 感到累 | "这只是身体的疲劳，不是真正的极限" |
| 想放弃 | "再坚持一个功能站，我就赢了" |
| 害怕 | "无数人做到了，我也可以" |
| 绝望 | "最难的已经过去了，接下来是下坡" |
| 疼痛 | "这个疼痛不会伤害我，我可以承受" |

### 分段目标法

```
🚩 心理崩溃时，重新设定目标：

❌ 原目标：还有8个功能站...（绝望）
✅ 新目标：只看下一个功能站

执行步骤：
1. 低头只看脚下10米
2. 走到下一个标志物（旗子/志愿者）
3. 告诉自己"我做到了"
4. 重复直到恢复状态
```

### 寻求外部支持
- **志愿者**：他们会鼓励你
- **队友/观众**：喊你的名字
- **手机记录**：录一句"我在坚持"给自己

## 六、紧急情况求助流程

### 紧急程度分级

| 级别 | 情况 | 应对 |
|------|------|------|
| 🟢 绿色 | 轻微不适、小伤 | 自行处理，继续比赛 |
| 🟡 黄色 | 中度不适、抽筋 | 暂停处理，求助志愿者 |
| 🔴 红色 | 严重伤痛、晕眩、心悸 | **立即停止，呼叫急救** |

### 紧急求助步骤

```
1️⃣ 大声呼救/举手示意
   ↓
2️⃣ 告知身边选手帮忙呼叫志愿者
   ↓
3️⃣ 清晰说明：位置 + 状况 + 人数
   示例："我是选手，我需要在A站得到帮助，
          我的同伴晕倒了"
   ↓
4️⃣ 等待时保持静止，不要再移动
   ↓
5️⃣ 服从医疗人员指挥
```

### 必须呼叫急救的情况
- 🚨 胸痛、胸闷、心悸
- 🚨 呼吸困难、无法喘气
- 🚨 意识模糊、晕倒
- 🚨 严重脱水、抽搐
- 🚨 骨折、关节脱位

### 赛后注意事项
- 比赛结束后**不要立即停止活动**
- 慢走10分钟让心率恢复
- 补充水分和碳水
- 如有不适**立即就医**
