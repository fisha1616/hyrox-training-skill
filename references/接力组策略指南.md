# HYROX接力组策略指南

## 四人接力组基础规则

- **总距离**：8km跑 + 8个功能站（与单人/双人组相同）
- **分工方式**：4人分跑跑步和功能站，每人负责2km跑 + 2个功能站
- **换人规则**：每完成1km跑 + 1个功能站后换下一位队友
- **完赛标准**：全队完成所有项目

---

## 组队建议

### 队友选择策略

**新手团队建议：找1-2个更强的队友**

| 队员配置 | 原因 |
|----------|------|
| 1-2个有经验/能力强的队友 | 负责高难度站点（推雪橇、弓步走、墙球） |
| 2-3个新手/体验者 | 负责相对轻松的站点（拉雪橇、农夫行走、波比跳） |

**为什么这样配置？**
- 高难度站点需要力量/技术基础，新手可能无法完成
- 有经验的队友可以"兜底"，确保团队顺利完赛
- 新手可以选轻松站点，体验为主

**避免的情况**：
- ❌ 4个都是新手 → 高难度站点无人能扛，可能无法完赛
- ❌ 水平差距太大+没有沟通 → 强者抱怨队友慢，体验差

### 陌生人组队（小红书/网络组队）

**最重要的不是合练，而是能在交接区认出队友**

| 识别方法 | 建议 |
|----------|------|
| **统一标识** | 同样颜色的护腕、发带、队服、帽子 |
| **提前沟通** | 确认每个人的站点分工、交接顺序 |
| **赛前见面** | 至少在签到时见一面，认脸+确认分工 |

**交接区识别清单**：
1. 约定统一的识别物（如：蓝色护腕）
2. 记住队友的长相/特征
3. 确认交接顺序（谁→谁→谁→谁）
4. 留下联系方式（微信/电话）

**陌生人组队注意事项**：
- 提前确认队友是否靠谱（是否真的参赛）
- 确认队友的训练水平和站点偏好
- 明确分工，避免临时争抢/推诿
- 合练不是必须，但赛前沟通是必须

## 出站顺序排序原则

### 核心原则：避免项目连续

**为什么重要？**
- 同一人连续项目会导致疲劳积累，影响后续表现
- 中间间隔可以让队员恢复体力、调整状态
- 团队整体节奏更均衡，减少"短板效应"

### 排序规则

| 规则 | 优先级 | 说明 |
|------|--------|------|
| ❌ 禁止连续 | 最高 | 同一人不能连续负责两个项目 |
| ⚠️ 间隔至少1个 | 高 | 同一人的两个项目之间至少隔1个项目 |
| ✅ 间隔2个最佳 | 推荐 | 同一人的两个项目之间最好隔2个项目（但不绝对） |

### 正确示例

| 顺序 | 站点 | 负责队员 | 说明 |
|------|------|----------|------|
| 1 | 滑雪机 | 板栗 | ✅ |
| 2 | 推雪橇 | 芒果 | ✅ |
| 3 | 拉雪橇 | Lumos | ✅ 与芒果隔开 |
| 4 | 波比跳远 | 板栗 | ✅ 与第1项隔2个 |
| 5 | 划船机 | Marco | ✅ |
| 6 | 农夫行走 | 芒果 | ✅ 与第2项隔3个 |
| 7 | 弓步走 | Lumos | ✅ 与第3项隔3个 |
| 8 | 墙球 | Marco | ✅ 与第5项隔2个 |

### 错误示例（需避免）

| 顺序 | 站点 | 负责队员 | 问题 |
|------|------|----------|------|
| 1 | 滑雪机 | 板栗 | - |
| 2 | 推雪橇 | 芒果 | - |
| 3 | 拉雪橇 | 芒果 | ❌ 连续项目，体力无法恢复 |
| 4 | 波比跳远 | Lumos | - |
| 5 | 划船机 | 板栗 | - |
| 6 | 农夫行走 | Marco | - |
| 7 | 弓步走 | Lumos | - |
| 8 | 墙球 | Marco | ⚠️ 只隔1个，不推荐 |

### 排序优化技巧

1. **先排每个人的第一个项目**：按站点顺序依次分配
2. **再排第二个项目**：确保与第一个项目间隔≥2
3. **检查冲突**：确认无连续、间隔合理
4. **微调**：根据队员专长和体力特点适当调整

## 交接区（TZ）规则详解

### 交接区基本规定

| 规则 | 说明 |
|------|------|
| 进入时机 | 接棒队员**只能在交接前片刻**进入 |
| 禁止逗留 | 此区域内禁止停留等待 |
| 识别责任 | 每支队伍需确保队友按时进入交接区 |

### 交接流程

1. 完成当前站点 → 跑入交接区
2. 与等候的队友**击掌交接**
3. 下一人开始跑步
4. 上一人**立即离开**交接区（避免拥堵）

### 特殊情况：背靠背完成

如果一名队员连续完成2次跑步+2个站点：
- **仍需**在第一个训练站点后跑经交接区
- 目的：确保计时芯片被"识别"
- 未遵守将自动触发处罚

### 交接区违规

| 违规行为 | 后果 |
|----------|------|
| 未按规定跑经交接区 | 自动罚时 |
| 通过"入口"拱门离开站点 | 2分钟罚时 |
| 交接区逗留造成拥堵 | 警告 |

## 换人区策略

### 换人时机

1. **提前沟通**：倒数第二项完成时，下一位队员做好准备
2. **快速交接**：换人区控制在10-15秒内
3. **情绪管理**：鼓励队友，传递正向能量

### 换人区禁忌

- ❌ 不要在跑步中途换人
- ❌ 不要在功能站未完成时换人
- ❌ 不要在换人区停留过长时间

---

## 训练建议

### 团队训练重点

1. **个人专项训练**：每人重点练自己负责的站点
2. **跑步训练**：每人至少能舒适完成2km跑
3. **换人演练**：模拟比赛换人流程
4. **团队计时**：定期进行全流程训练

### 推荐训练频率

- 每周2-3次专项训练
- 每周1次团队合练
- 赛前2周进行1-2次完整模拟

## 常见问题

**Q: 可以临时调整站点分配吗？**
A: 可以，但建议赛前确定并充分训练，临时调整可能影响团队配合。

**Q: 如果有队友受伤怎么办？**
A: 接力组无法替补，建议赛前充分准备，避免受伤。如比赛中受伤，团队可以选择DNF（未完赛）或让其他队员分担更多任务。

**Q: 接力组有团队成绩吗？**
A: 有，团队成绩=所有成员完成时间总和，换人时间计入总时间。
