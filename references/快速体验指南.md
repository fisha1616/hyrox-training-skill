## 🎯 什么是快速体验模式？

让你在**7天内**感受HYROX的核心魅力：
- 🏃 8次跑步片段（每次1km）
- 🏋️ 4个经典站点入门
- ⏱️ 掌握比赛节奏

**不需要任何测试，不需要设定目标，马上开始！**

## 📋 每日详细说明

### Day 1：起点认知
```
🏃 跑步：3km，配速舒适（能说话）
📖 站姿学习：观看1分钟墙球动作示范
💡 目标：感受"跑完还能动"的节奏
```

### Day 2：站点初探
```
🏋️ 墙球：3kg药球，20次，体验发力
🧘 核心：平板支撑3×30秒
💡 目标：找到"髋铰链"发力感觉
```

### Day 3：节奏切换
```
🏃 间歇：400m快跑 × 6组，组间走200m
🔄 过渡练习：跑完立刻做10个深蹲
💡 目标：适应跑步-动作切换的喘不上气
```

### Day 4：主动恢复
```
🧘 拉伸：重点髋屈肌、腘绳肌
🫧 泡沫轴：大腿前侧、后侧各2分钟
💡 目标：让身体恢复，为后半程准备
```

### Day 5：力量入门
```
🏋️ 农夫行走：双手各6kg哑铃，走100m
🦵 深蹲：徒手20个，节奏放慢
💡 目标：感受"累到握不住"的手部疲劳
```

### Day 6：迷你模拟 ⭐
```
这是体验的核心！
---
🏃 跑 1km（热身配速）
🏋️ 墙球 30次（3kg）
🏃 跑 1km（稍快）
🏋️ 农夫行走 100m（6kg哑铃）
---
💡 目标：完整感受"跑-站-跑"的节奏崩溃曲线
```

### Day 7：收尾复盘
```
📹 必看：YouTube搜索"HYROX Official Event Recap"
📝 自问：
  - 跑步后还能做动作吗？
  - 握力最先崩溃还是腿部先酸？
  - 哪个动作最让你畏惧？
```

## ⚡ 常见问题

**Q：完全没练过，能跟下来吗？**
> A：Day 1-2用轻松配速，Day 3可以减少到4组，Day 6的墙球降到20次。**量力而行，安全第一。**

**Q：没有墙球/哑铃怎么办？**
> A：用书包装重物替代、水桶装水、或者只用徒手动作。效果打8折，但能体验核心。

**Q：7天后想继续，该怎么做？**
> A：进入完整备赛流程 → [查看入门路径图](./入门路径图.md)
