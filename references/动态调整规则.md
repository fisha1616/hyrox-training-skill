# HYROX 动态调整规则

> 训练不是死板执行，是根据身体反馈实时调整的艺术。本文档提供量化规则和决策流程，确保训练效果最大化、伤病风险最小化。

---

## 一、训练负荷评估方法

### 1.1 RPE量表（主观疲劳感觉）

| RPE | 等级 | 感觉描述 | 对应强度 |
|-----|------|----------|----------|
| **1** | 极轻松 | 几乎不费力，像休息 | 恢复性活动 |
| **2** | 很轻松 | 感觉很轻松，能一直做 | 轻松热身 |
| **3** | 轻松 | 不太费力，感觉舒适 | 轻松有氧 |
| **4** | 较轻松 | 有点费力，但能坚持 | 节奏跑 |
| **5** | 中等 | 中等费力，呼吸加深 | 有氧阈值 |
| **6** | 较费力 | 感觉较吃力，需努力 | 阈值训练 |
| **7** | 费力 | 感觉很吃力，但能完成 | 强度训练 |
| **8** | 很费力 | 非常吃力，勉强完成 | 高强度 |
| **9** | 极费力 | 极度吃力，速度下降 | 冲刺训练 |
| **10** | 力竭 | 完全力竭，无法继续 | 全力输出 |

### 1.2 每日晨脉检测

| 指标 | 正常范围 | 预警信号 | 处理建议 |
|------|----------|----------|----------|
| **晨脉** | 个人基准±5bpm | 连续3天升高>10% | 减量30-50% |
| **静息心率** | 55-70bpm | 突然升高5bpm+ | 休息+检查 |
| **体感** | 正常 | 明显疲劳/酸痛 | 降低强度 |

> **基准建立**：连续测量7天晨脉，取平均值作为个人基准

### 1.3 训练后恢复评估

| 问题类型 | 表现 | 原因 | 调整措施 |
|----------|------|------|----------|
| **心肺过度** | 训练后30min心率仍>100 | 强度过大 | 次日降强度+加恢复 |
| **肌肉过度** | 次日酸痛明显 | 训练量过大 | 减量或换低冲击动作 |
| **关节不适** | 关节隐痛/肿胀 | 动作问题或过度 | 停止相关动作，查原因 |
| **睡眠问题** | 入睡困难/早醒 | 压力过大 | 减量+放松训练 |

## 二、量化调整规则

### 2.1 心率异常调整规则

| 检测结果 | 原因分析 | 调整措施 | 执行时间 |
|----------|----------|----------|----------|
| **训练中持续偏高**（>90%最大心率） | 强度过高/恢复不足 | 次日训练降强度20% | 立即执行 |
| **训练中无法达标**（<70%目标心率） | 状态差/热身不够 | 延长热身10分钟 | 本次训练 |
| **心率恢复慢**（5分钟后>100bpm） | 心肺疲劳/疾病前兆 | 次日休息或极轻度 | 立即执行 |
| **晨脉升高>10%** | 身体未恢复 | 休息1天+减量30% | 次日执行 |

**计算示例**：
```
目标心率区间：140-160bpm（最大心率180×78-89%）
实际心率：持续165bpm+
调整后强度：原计划80%→60%
```

---

### 2.2 动作质量差调整规则

| 问题表现 | 原因分析 | 调整措施 | 后续跟进 |
|----------|----------|----------|----------|
| **技术动作变形** | 疲劳/肌肉力量不足 | 加练分解动作 | 2-3天后原动作复查 |
| **无法完成计划次数** | 力量/耐力不足 | 减量至能完成70% | 3天后重新评估 |
| **出现代偿动作** | 疲劳/伤病前兆 | 停止，替换动作 | 次日检查伤病 |
| **协调性下降** | 神经疲劳 | 减量+简化训练 | 充分休息后再练 |

**分解训练补充方案**：
```
原动作：Wall Balls 20次×5组
调整后：
  - 分解1：深蹲 10次（建立发力）
  - 分解2：肩上推举 10次（建立推力）
  - 分解3：完整Wall Balls 10次×3组（逐步整合）
```

---

### 2.3 伤病/疲劳处理规则

| 伤病/疲劳类型 | 替换动作建议 | 注意事项 |
|--------------|--------------|----------|
| **膝关节不适** | 深蹲→腿举/腿弯举 | 停止跳跃类动作 |
| **肩关节不适** | 推举→侧平举/面拉 | 避免过头动作 |
| **下背不适** | 硬拉→臀桥/背伸展 | 核心训练减量 |
| **手腕不适** | 农夫行走→坐姿搬运 | 减少抓握类动作 |
| **小腿酸痛** | 跑步→椭圆机/游泳 | 低冲击有氧替代 |
| **全身疲劳** | 高强度→瑜伽/拉伸 | 完全休息1-2天 |

**伤病恢复后回归流程**：
```
第1阶段（3天）：完成计划50%，RPE≤5
第2阶段（3天）：完成计划75%，RPE≤6
第3阶段（1周）：完成计划100%，RPE≤7
第4阶段：恢复正常训练
```

---

### 2.4 缺训处理规则

| 缺训情况 | 处理原则 | 弥补策略 |
|----------|----------|----------|
| **缺1天** | 计划顺延 | 次日完成当日计划即可 |
| **缺2天** | 精简非核心 | 只做核心训练（跑步+薄弱站） |
| **缺3-4天** | 减量30-50% | 恢复后首周降低强度 |
| **缺1周+** | 重新评估 | 延长备赛周期或调整目标 |
| **连续缺训** | 计划重排 | 根据剩余时间重新规划 |

**缺训后恢复训练原则**：
- ✅ 先做低强度有氧激活
- ✅ 训练量恢复至正常70%
- ✅ 强度恢复至正常80%
- ✅ 3天内不尝试高强度训练
- ❌ 不"补"训练——身体需要恢复

## 三、疲劳预警机制

### 3.1 连续高疲劳强制恢复

| 预警等级 | 触发条件 | 强制措施 |
|----------|----------|----------|
| **黄色预警** | 连续2天RPE≥8 | 次日强制休息，训练减量50% |
| **橙色预警** | 连续3天RPE≥8 或 晨脉升高>10% | 休息2天+主动恢复训练 |
| **红色预警** | 连续4天RPE≥8 或 出现伤痛 | 休息3天，必要时就医 |

### 3.2 疲劳自检清单

**每日训练前完成**：

| 检查项 | 正常 | 异常 |
|--------|------|------|
| 睡眠质量 | ≥6小时，感觉休息好 | 失眠/早醒/疲惫 |
| 晨起状态 | 身体轻松 | 酸痛/僵硬/疲惫 |
| 关节感觉 | 无不适 | 隐痛/肿胀/卡顿 |
| 训练意愿 | 有动力 | 极度抗拒 |
| 心率 | 正常 | 明显加快 |

**判断标准**：
- ≥3项异常 → 降低强度或休息
- 出现关节异常 → 停止训练，检查原因
- 心理极度抗拒 → 休息，这是疲劳信号

## 五、特殊情况处理

### 5.1 女性生理期处理

| 阶段 | 身体状态 | 训练建议 |
|------|----------|----------|
| **经期1-2天** | 量大/痛经 | 休息或轻度活动（散步/拉伸） |
| **经期3-5天** | 量减少/恢复中 | 低强度有氧+上肢训练 |
| **经期后** | 身体恢复 | 逐步恢复正常训练强度 |

### 5.2 天气极端情况

| 情况 | 影响 | 调整措施 |
|------|------|----------|
| **高温（>30°C）** | 心率升高、出汗多 | 缩短训练时长，补充电解质 |
| **低温（<5°C）** | 肌肉僵硬 | 延长热身，注意保暖 |
| **雨天/潮湿** | 器械湿滑 | 做好防滑，注意热身 |
| **雾霾** | 呼吸不适 | 改为室内训练 |

### 5.3 赛前一周调整

| 时间 | 训练内容 | 强度 |
|------|----------|------|
| 比赛前7天 | 正常训练 | 80% |
| 比赛前6天 | 正常训练 | 70% |
| 比赛前5天 | 正常训练 | 60% |
| 比赛前4天 | 训练 | 50% |
| 比赛前3天 | 轻度激活 | 30% |
| 比赛前2天 | 完全休息/拉伸 | 0% |
| 比赛前1天 | 5分钟轻松跑+拉伸 | 20% |
| **比赛日** | **全力以赴** | **100%** |

## 七、核心原则速记

| 原则 | 说明 |
|------|------|
| **感觉优先** | 身体感觉比计划更重要 |
| **宁可少做** | 状态差时减量，好过受伤停训 |
| **记录反馈** | 每次训练都记录，便于分析调整 |
| **渐进超负荷** | 增量和增强度都要循序渐进 |
| **休息也是训练** | 恢复期身体在适应和强化 |
| **个性化调整** | 别人的计划只是参考，自己的身体才最诚实 |
