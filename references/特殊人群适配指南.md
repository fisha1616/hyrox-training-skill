## 伤病替代动作库

> ⚠️ **重要提示**：以下替代动作仅供参考，受伤后请务必遵医嘱，康复后再逐步恢复训练。

### 膝盖问题

| 原动作 | 症状表现 | 推荐替代 | 替代理由 |
|--------|----------|----------|----------|
| **弓步走** | 膝盖疼痛/不适 | 踏步上箱 | 减少膝盖冲击 |
| **波比跳** | 膝盖刺痛 | 简化波比（不跳） | 消除跳跃冲击 |
| **农夫行走** | 膝盖压力感 | 坐姿握重 | 避免负重移动 |
| **推雪橇** | 膝盖无法承重 | 手推阻力撬 | 改用上肢发力 |

**膝盖康复训练建议：**
- 每天2组靠墙静蹲（每次60秒）
- 每天10分钟骑单车（低阻力）
- 游泳/椭圆机替代跑步

### 肩部问题

| 原动作 | 症状表现 | 推荐替代 | 替代理由 |
|--------|----------|----------|----------|
| **墙球** | 肩袖疼痛/撞击感 | 过头推举 | 改变发力角度 |
| **滑雪机** | 肩部不适 | 坐姿划船 | 减少肩关节负担 |
| **推雪橇** | 推举时疼痛 | 俯卧撑系列 | 改用胸背发力 |
| **划船机** | 拉桨时不适 | 动感单车 | 完全避开上肢 |

**肩部康复训练建议：**
- 肩部绕环热身（每天5分钟）
- 弹力带肩外旋（每天3组×15次）
- 游泳（仰泳为主）替代高强度训练

### 腰背问题

| 原动作 | 症状表现 | 推荐替代 | 替代理由 |
|--------|----------|----------|----------|
| **推雪橇** | 下背疼痛/僵硬 | 阻力撬行走 | 改变身体角度 |
| **农夫行走** | 腰部不适 | 坐姿负重转体 | 避免脊柱承重 |
| **硬拉系列** | 腰椎间盘问题 | 臀桥/超人不倒翁 | 保护腰椎 |
| **波比跳** | 弯腰时疼痛 | 简化波比（不弯腰） | 保持脊柱中立 |

**腰背康复训练建议：**
- 每天10分钟死虫式（Dead Bug）
- 平板支撑（每天3组，每组30秒）
- 麦肯基疗法缓解腰痛

### 通用替代原则

1. **减少冲击** → 跳→走，走→站
2. **改变角度** → 高位→低位，垂直→水平
3. **替代肌群** → 受伤部位→代偿部位
4. **降低负荷** → 减重→不负重

## 女性训练调整

### 经期训练建议

| 阶段 | 身体状态 | 训练调整 |
|------|----------|----------|
| **经期1-3天** | 腹痛/乏力 | 休息或仅做轻量拉伸 |
| **经期4-5天** | 逐渐恢复 | 降低30%强度，避免核心+下肢高强度 |
| **经期后一周** | 状态最佳 | 可正常训练，可作为比赛模拟周 |
| **经期前一周** | 情绪波动/腹胀 | 降低训练量10-20% |

**经期训练原则：**
- ✅ 可以训练，但降低强度
- ✅ 以有氧为主（跑步/单车）
- ❌ 避免瓦氏呼吸（憋气发力）
- ❌ 避免倒立/过头动作多的训练

**经期备赛建议：**
- 记录周期，合理安排高强度训练日
- 比赛日如遇经期，可使用止痛药+保暖
- 准备备用衣物和热水

### 产后恢复训练

**产后恢复阶段划分：**

| 产后时间 | 可进行 | 禁止动作 |
|----------|--------|----------|
| 0-6周 | 呼吸练习/轻柔步行 | 所有功能性训练 |
| 6-12周 | 盆底肌训练/核心感知 | 卷腹/仰卧起坐 |
| 3-6个月 | 低强度有氧（走路/慢跑） | 高冲击动作 |
| 6个月+ | 逐步恢复功能性训练 | 循序渐进 |

**恢复重点：**
1. **盆底肌训练**（凯格尔运动）—— 每天3组×10次
2. **腹直肌分离检测**—— 分离>2指先修复再训练
3. **核心稳定性**—— 死虫式/鸟狗式优先
4. **逐步递增**—— 每周增加不超过10%强度

**产后参赛建议：**
- 顺产建议6个月后参赛
- 剖腹产建议12个月后参赛
- 参赛前进行医生评估

## 40+年龄组

### 关节保护建议

| 关节 | 保护措施 | 训练调整 |
|------|----------|----------|
| **膝盖** | 训练前充分热身 | 避免深幅弓步 |
| **肩袖** | 每天肩部绕环 | 减少大重量推举 |
| **腰椎** | 硬拉佩戴护腰 | 避免脊柱屈曲负重 |
| **髋关节** | 髋关节灵活性训练 | 增加热身时间 |

**每日关节保护routine（10分钟）：**
```
1. 膝盖绕环 × 10圈
2. 肩部绕环 × 10圈
3. 髋关节绕环 × 10圈
4. 脊柱扭转 × 5组
5. 踝关节绕环 × 10圈
```

### 训练强度调整

| 指标 | 30岁以下 | 40-50岁 | 50岁以上 |
|------|----------|---------|----------|
| **周训练量** | 100% | 80% | 60-70% |
| **单次时长** | 60-90分钟 | 45-60分钟 | 30-45分钟 |
| **高强度占比** | 40-50% | 30-40% | 20-30% |
| **组间休息** | 60-90秒 | 90-120秒 | 120-180秒 |

**心率控制建议：**
- 最大心率 ≈ 220 - 年龄
- 训练心率上限 = 最大心率 × 85%
- 40-50岁：靶心率区间 120-144 bpm
- 50岁以上：靶心率区间 108-136 bpm

### 恢复时间延长

| 恢复项目 | 30岁以下 | 40-50岁 | 50岁以上 |
|----------|----------|---------|----------|
| **肌肉酸痛** | 24-48小时 | 48-72小时 | 72-96小时 |
| **高强度训练间隔** | 48小时 | 72小时 | 96小时+ |
| **睡眠需求** | 7-8小时 | 7-8小时 | 8-9小时 |
| **营养补充** | 常规 | 增加蛋白质 | 蛋白质+氨基葡萄糖 |

**加速恢复建议：**
1. **睡眠**：每晚8小时，午休20分钟
2. **营养**：蛋白质1.2-1.5g/kg体重
3. **主动恢复**：训练后散步30分钟
4. **冷热交替**：冷水澡+热水浴促进循环

### 40+参赛注意事项

| 项目 | 建议 |
|------|------|
| **热身** | 比年轻人多花5-10分钟热身 |
| **赛中补水** | 每15分钟补充水分，不要等渴了再喝 |
| **赛中补给** | 提前吃能量胶，不要空腹运动>60分钟 |
| **赛后恢复** | 拉伸30分钟+充足睡眠 |

## 特殊人群参赛检查清单

### 伤病选手

- [ ] 医生确认可以参赛
- [ ] 已准备替代动作方案
- [ ] 比赛当天携带相关药物
- [ ] 了解最近的医疗站点位置
- [ ] 已告知团队成员自己的伤病情况

### 女性选手

- [ ] 已记录经期周期
- [ ] 比赛日已准备生理期用品
- [ ] 了解经期训练强度调整原则
- [ ] 产后选手已获医生参赛许可

### 40+选手

- [ ] 已评估自身关节健康状况
- [ ] 训练强度已做适应性调整
- [ ] 比赛补给已准备充分
- [ ] 已安排充足的赛后恢复时间
