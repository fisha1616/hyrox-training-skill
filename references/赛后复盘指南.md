## 成绩预测模型说明

### 基于训练数据的完赛时间预测

**预测公式：**

```
预测完赛时间 = Σ(各站点训练用时 × 比赛系数) + Σ(跑步训练配速 × 距离) + 缓冲时间

比赛系数：
- 滑雪机：1.0（训练时间≈比赛时间）
- 划船机：1.0
- 推雪橇：1.05（比赛压力+5%）
- 拉雪橇：1.05
- 墙球：1.1（心理压力+10%）
- 波比跳：1.1
- 弓步走：1.05
- 农夫行走：1.0

缓冲时间：5-10分钟（根据个人经验调整）
```

### 使用建议

1. **训练记录**：每次训练记录各站点的完成时间
2. **周期对比**：每周对比训练数据，观察进步趋势
3. **赛前预测**：比赛前1周用最新训练数据预测完赛时间
4. **动态调整**：如果训练用时有突破，相应调整预测

**注意**：预测模型仅供参考，实际比赛受状态、天气、赛道等因素影响。
